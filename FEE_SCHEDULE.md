## Google Form Setup Guide

### Recommended Form Fields:

1. **Student Name** (Short answer)
   - Required

2. **Student Class** (Multiple choice)
   - Preschool, Nursery 1, Nursery 2, Nursery 3
   - Basic 1, Basic 2, Basic 3, Basic 4, Basic 5
   - JSS1, JSS2, JSS3
   - SS1, SS2

3. **Family Name** (Short answer)
   - Optional

4. **Tuition Fee Paid** (Number)
   - Optional

5. **Exam Fee Paid** (Number)
   - Optional

6. **Transport Fee Paid** (Number)
   - Optional

7. **Development Levy Paid** (Number)
   - Optional

8. **Stationery Fee Paid** (Number)
   - Optional

9. **Other Fees/Notes** (Paragraph)
   - Optional

10. **Date of Payment** (Date)
    - Optional

---

## How to Create the Google Form

1. Go to [forms.google.com](https://forms.google.com)
2. Create a new form
3. Add the fields listed above
4. Use the fee schedule above as reference values
5. Set responses to go to a Google Sheet
6. Download responses as CSV/Excel
7. Import into the web app using the upload feature
